# Snake_Python

### Description:
A simple snake type game made in pygame zero. Just for fun. You can adjust map size by two parameters:


- CELL = 25   #size of cell<br>
- ROW_X = 20  #horizontal size<br>
- ROW_Y = 20  #vertical size

  
### Requirements:

python -m pip install pgzero
